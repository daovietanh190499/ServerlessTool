Hướng dẫn

Script của bạn phải chứa một hàm process(input_data) nhận một tham số đầu vào và trả về kết quả.

Ví dụ:

def process(input_data):
    # Xử lý dữ liệu đầu vào
    result = input_data.upper()  # Chuyển đổi thành chữ hoa
    return result
